# 🏥 Análise de Dados Hospitalares com Python

Projeto de análise exploratória de dados (EDA) de um hospital fictício, desenvolvido para praticar manipulação e visualização de dados com Python.

## 📊 Sobre o projeto

Este projeto realiza uma análise completa de dados hospitalares, abrangendo pacientes, consultas, internações e estoque de medicamentos. Os dados foram gerados sinteticamente para fins educacionais.

## 🔍 Análises realizadas

- Distribuição de idades, gênero e tipo sanguíneo dos pacientes
- Faturamento por médico e por especialidade
- Evolução mensal das consultas
- Custo total por motivo de internação
- Duração das internações por categoria
- Controle e alertas de estoque de medicamentos

## 🛠️ Tecnologias utilizadas

- Python 3
- Pandas
- Matplotlib
- Seaborn
- NumPy

## 📁 Estrutura do projeto

```
├── hospital_dados.xlsx        # Base de dados
├── analise_hospital.ipynb     # Notebook com toda a análise
└── hospital_relatorio_final.xlsx  # Relatório exportado
```
